## 文档管理

### 上传文档

**POST** `/api/documents/upload`

上传文档文件到知识库

**请求方式**: `POST`

**Content-Type**: `multipart/form-data`

**请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| file | binary | 是 | 文档文件 |

**响应示例**:

```json
{
  "success": true,
  "file_id": "string",
  "filename": "string",
  "message": "string"
}
```

---

### 解析文档

**POST** `/api/documents/parse`

解析上传的文档并生成向量

**请求方式**: `POST`

**Content-Type**: `application/json`

**请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| file_id | string | 是 | 文件ID |
| filename | string | 是 | 文件名 |
| department | string | 否 | 部门 |
| regulation_type | string | 否 | 制度类型 |
| tags | array[string] | 否 | 标签列表 |
| effective_date | string | 否 | 生效日期 |
| source | string | 否 | 来源 |

**请求示例**:

```json
{
  "file_id": "abc123",
  "filename": "制度文件.pdf",
  "department": "研发部",
  "regulation_type": "管理制度",
  "tags": ["重要", "内部"],
  "effective_date": "2024-01-01",
  "source": "内部制定"
}
```

**响应示例**:

```json
{
  "success": true,
  "file_id": "string",
  "message": "string"
}
```

## 检索功能

### 智能检索

**POST** `/api/search`

支持语义检索、关键词检索和混合检索

**请求方式**: `POST`

**Content-Type**: `application/json`

**请求参数**:

| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| query | string | 是 | - | 检索查询语句 |
| top_k | integer | 否 | 5 | 返回结果数量 |
| search_type | string | 否 | hybrid | 检索类型（semantic/keyword/hybrid） |

**请求示例**:

```json
{
  "query": "请假制度的相关规定",
  "top_k": 10,
  "search_type": "hybrid"
}
```

**响应**: 检索结果列表

## 错误响应

当请求参数验证失败时，返回 422 状态码：

```json
{
  "detail": [
    {
      "loc": ["body", "field_name"],
      "msg": "错误消息",
      "type": "error_type"
    }
  ]
}
```
